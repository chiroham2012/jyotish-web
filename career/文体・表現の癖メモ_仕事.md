# 文体・表現の癖メモ（仕事鑑定文）

## 1. 文体ルール（守ってほしい癖）

- **占星術の内部用語を本文に出さない。**
  - 「材料」「材料ボード」「★」などの裏方の言葉はお客様向け本文に書かない。
  - 例：×「その火星が、材料の中でもっとも強く出ています」 → ○「その火星の影響がもっとも強く出ています」
- **メタな前置き・説明的な導入を置かない。**
  - 「中心を支える材料がいくつもあります。まず〜」のような、これから何を書くかの前振りは削り、いきなり中身から入る。
  - 例：×「中心を支える材料がいくつもあります。まず先ほどの強い火星が、」 → ○「先ほどの強い火星が、」
- **冗長な助詞を削り、語尾をすっきりさせる。**
  - 例：×「経験しやすいのかもしれません」 → ○「経験しやすいかもしれません」
- **否定の言い回しは、固い断定調より自然で柔らかい言い切りにする。**
  - 例：×「『向いていない』という意味ではないと考えられます」 → ○「『向いていない』という意味とは限りません」
- **該当する象意は、お客様にイメージしやすい具体語で足してよい。**
  - 例：獅子座10室 →「中心に立って率いる」だけでなく「芸能・歌やダンスなど目立つ仕事」も添える。
- **専門用語は本文では平易な言い方に言い換える。**（§1冒頭「内部用語を出さない」の具体化。室番号やサイン名は出してよいが、品位・支配の専門語は崩す）
  - 例：×「太陽は獅子座で定座となり」 → ○「太陽の強い影響から」
  - 例：×「土星は12室にあって力を出しにくいサインに位置し」 → ○「土星は12室にあって力を出しにくい配置で」
  - 例：×「太陽に支配される場所です」 → ○「太陽の影響の強い1室です」
- **「○室を司る○○星」より、その星が実際にいるサイン名で名指す。**
  - 例：×「この10室を司る金星は11室にあり」 → ○「牡牛座に縁ある金星は双子座にあり、11室にいます」
- **「〜のような場所です」と説明で締めず、その部屋・配置を直接名指して言い切る。**
  - 例：×「人を率いていくような場所です」 → ○「人を率いていく太陽の影響の強い1室です」
- **強い惑星（shadbala強い）は「強い働き」だけで終えず、人の資質・行動に翻訳して具体化してよい。**（同語反復も防げる）
  - 例：火星が強い →「強く力を発揮しているので、それを成そうとする情熱やエネルギーも高く持っています」
  - 例：凶星アスペクト →「バランスが崩れた時のやり過ぎ・言い過ぎに注意」など、行動面の留意に翻訳。
- **過剰な懇願・お願い表現を避ける。**
  - 例：×「どうかご自身の強みとして大切にされてください」 → ○「ご自身の強みとして大切にされてください」
- **同じ言い回しを近接して繰り返さない（同語反復を散らす）。**
  - 例：「しっかりと力を持って働いています」を器・火星・木星で3連続 → 2つ目以降は「強い力をもった火星があります」「強い働きを見せています」等に変える。
- **サインの性質は「〜星座です」「〜性質をもっています」と言い切る。**「〜色をもった場所です」のような説明的な体言止めにしない。
  - 例：×「学びや真理を広めていこうとする色をもった場所です」 → ○「学びや真理を広めていこうとする星座です」
- **職業表示星の太陽の枕詞は「社会での姿を表す太陽」。**「職業を自然にあらわす太陽」より、お客様にイメージしやすい。
- **重い語尾「〜と考えられます」を連続させない。**段落末に並んだら一部を「〜でしょう」など軽い言い切りに散らす（§1「同語反復」と同じ方針）。
  - 例：×各段落の末尾が「〜と考えられます」で続く → 一部を「〜中心にあるでしょう」等に。
- **指示語より具体語。**「それを成そうとする」→「何かを成そうとする」。
- **ひらがなの一部は漢字に寄せる（佐藤さんの好み）。**「あわせ→合わせ」「かたち→形」「ぶん→分」「とくに→特に」「力みすぎ→力み過ぎ」。
- **冒頭の指示語・重い修飾語を削り、主語からすっと入る。**（§1「メタ前置きを置かない」の語レベル版）
  - 例：×「このケートゥの影響は」 → ○「ケートゥの影響は」
  - 例：×「職業そのものを象徴する土星が」 → ○「土星が」
  - 例：×「しかも強く働いています」 → ○「強く働いています」
- **文末の「〜ことを示しています」「〜と読めます」を避け、直接言い切る。**（§1「重い語尾を散らす」「言い切り」の具体化）
  - 例：×「王道のキャリアになりやすいことを示しています」 → ○「王道のキャリアになりやすいでしょう」
  - 例：×「適性があると読めます」 → ○「適性があります」
- **直前で名指した部屋・サインは繰り返さず「そこ」で受ける／見出し語と同じ語を本文で重ねない（別語に）。**（§1「同語反復を散らす」の具体化。「指示語より具体語」の例外＝直前に出た対象は指示語で受けてよい）
  - 例：×「まさにその10室に在住し、自分の星座（双子座）で力を発揮」 → ○「まさにそこに在住し、いきいきと力を発揮できる配置です」
  - 例：（見出し「あなたという器」に対し）×「乙女座の器や」 → ○「乙女座の質や」
- **硬い・比喩的な動詞は能動で平易な動詞に。**
  - 例：×「しっかり力をもっています」 → ○「しっかり力を発揮しています」
  - 例：×「なほこさん自身（1室）を照らしています」 → ○「なほこさん自身を表す1室に影響を与えています」
- **「〇室の△△がもう一つ別の場所（□室）に位置しているため」のような回りくどい二重指示は避け、「〇室にあって△△な配置で」と直接言い切る。**（占星術に不慣れな読み手には「もう一つ別の場所」が何を指すか伝わらない）
  - 例：×「この10室の月がもう一つ別の場所（12室）に位置しているため」 → ○「仕事の場を司る月は12室にあって力を出しにくい配置で」
